浏览器插件推荐

发表于 2020-02-17 分类于 浏览器插件
本文字数： 626 阅读时长 ≈ 1 分钟

今天推荐几款chrome浏览器(也Chromium内核的浏览器)，插件绝对好用！！！不多废话，直接上干货

1.ABP广告拦截器

这款拦截器特别好用，用过的人肯定知道，废话少说直接上演示对比图：

之前：

之后：

可以看到广告都没有了，可以帮你避免不必要的麻烦，是不是很good。

2.Google翻译

当你看到一段不会的英文，可以使用Google翻译来帮助你翻译它，选择然后会出来一个Google翻译的logo直接点击

3. IDE下载器

多通道下载技术，让你下载更快。注意：支持某网盘下载哦！

4.Infinity 新标签页 (Pro)

强烈推荐使用，这款插件非常非常非常好用，用了回不去的那种

直接上图：

设计非常简洁、好看，添加插件后他会自动给你添加一些常用的网站，直接在下面圆形图标里找就可以，你也可以点击右上角无限符号添加、自定义自己喜欢的网页，如果你不想要某些网页直接右击删除就可以，很方便很好操作

你还可以点击右下角小风车来更换它的图片背景

5.Smallpdf - 编辑、压缩与转换PDF文件

它可以提高你的办公效率，支持PDF转PPT、Excel转PDF、PDF转world等等格式

6.油猴

油猴可以添加向浏览器添加一些脚本文件实现很多有意思的功能

结束语

这些就是我推荐的几款好用的chrome插件，这些插件在百度直接可以找到，如果不想找也可以直接私信我要。虽然不多但都非常实用，希望大家喜欢！！！

最后希望大家多多关注我的博客：srihzz.ml，后续我会更新更多有趣、实用的内容！
